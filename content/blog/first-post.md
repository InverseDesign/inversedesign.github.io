## 主要特点

### 简洁优雅的设计
Hextra 主题提供了现代化的设计，支持亮色和暗色模式。

### 响应式布局
网站在各种设备上都能完美显示，从手机到桌面。

### 内置搜索功能
使用 FlexSearch 提供快速的全文搜索体验。

### 丰富的组件
提供多种短代码组件，如 callout、tabs 等。

## 文件结构

```
content/
├── _index.md          # 主页
├── about.md           # 关于页面
├── blog/              # 博客目录
│   ├── _index.md      # 博客首页
│   └── first-post.md  # 本文
└── docs/              # 文档目录
    ├── _index.md      # 文档首页
    ├── tech/          # 技术文档
    └── learning/      # 学习资源
```
